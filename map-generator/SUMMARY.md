## How to Use the Map Generator

1. Open `index.html` or `combined-demo.html` in your web browser
2. Upload your data file (CSV or Excel) with state codes and values
3. Customize the map appearance using the sidebar options
4. Copy the generated embed code
5. Paste the embed code into your Shopify store or any other website

## Technical Details

- **Frontend**: HTML, CSS, JavaScript
- **Libraries**: D3.js, TopoJSON, SheetJS, PapaParse
- **Data Storage**: LocalStorage for saving map configurations
- **Embedding**: iframe-based embedding with responsive design
